Mini Hub - Exercices IA Mails

Objectif de l'exercice

Le but de cet exercice est d'analyser des mails fictifs en ASCII. Vous devrez développer un prompt qui permet d'extraire efficacement les informations pertinentes des mails fournis.
Voir un exemple de prompt (solution)

Tu es un assistant spécialisé dans l'analyse de mails ASCII. Pour chaque mail que je t'envoie, extrais les informations principales et présente-les dans un tableau clair avec les colonnes suivantes :

  • Expéditeur
  • Destinataire
  • Date
  • Objet
  • Corps du message résumé

Analyse uniquement les informations présentes dans le mail, ne complète rien d'inventé. Sois synthétique dans le résumé.
