Présentation du projet

Duerp est un outil conçu pour simplifier et automatiser la collecte d'information en vue de remplir le DUER de l'établissement.

Les formulaires

Voici 3 versions de ce formulaire, à adapter selon vos besoins :

  • formulaireV2.html - Formulaire interactif pour l'identification des risques avec des catégories dynamiques.
    Voir le fichier
  • formulairecsv.html - Formulaire permettant de générer un fichier CSV des risques identifiés.
    Voir le fichier
  • formulaire.html - Formulaire d'évaluation des risques avec une notice explicative et une option d'impression.
    Voir le fichier
